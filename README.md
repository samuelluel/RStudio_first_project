📊 Statistical Learning Coursework (DAR)

📌 Overview

This project contains my coursework for the Data Analytics (DAR) module at Birkbeck.
The work focuses on applying statistical learning methods using R, including regression analysis and interpretation of results.

⸻

🎯 Objectives
 • Understand key statistical learning concepts
 • Apply linear regression models in R
 • Interpret model outputs (coefficients, p-values, R²)
 • Visualise relationships between variables
 • Demonstrate practical data analysis skills

⸻

🧠 Methods Used
 • Linear Regression (lm in R)
 • Hypothesis Testing (t-test, p-value interpretation)
 • Exploratory Data Analysis (EDA)
 • Data visualisation (scatter plots, fitted line)

⸻

📂 Files Included
 • coursework.Rmd → Main analysis (R Markdown)
 • coursework.pdf → Final knitted output
 • data.csv → Dataset

⸻

📈 Key Results
 • Found a significant relationship between variables
 • Slope coefficient shows how one variable changes with another
 • p-value < 0.05 → statistically significant
 • Model explains a good portion of variability (R²)

⸻

🛠️ Tools & Technologies
 • R
 • RStudio
 • R Markdown

⸻

💡 What I Learned
 • How regression models work in practice
 • How to interpret statistical outputs clearly
 • Importance of data visualisation
 • Writing reproducible analysis using R Markdown

⸻

🚀 How to Run
 1. Open the .Rmd file in RStudio
 2. Click Knit to generate the PDF
 3. Make sure required packages are installed
